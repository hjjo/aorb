# Aorb

## Backend

```json
# create virtual env
virtualenv -p python3 env

# activate env
source env/bin/activate

# install packages
pip install -r requirements.txt

# db migrate
python manage.py migrate

# run server
python manage.py runserver
```

## Frontend

```json
# Installation
npm i

# Run
npm start

# build
npm run build
```
